ArgoCD — Déploiements GitOps

ArgoCD est le moteur GitOps du cluster KubeSecureBox. Il synchronise l’état du cluster avec les dépôts Git définis (applications, Helm charts, manifests Kubernetes) et assure le déploiement continu.

Chaque modification poussée dans les dépôts configurés peut être appliquée automatiquement ou manuellement. Les options sync auto et self-heal permettent de maintenir le cluster aligné avec la source de vérité et de corriger les dérives.

Accès

L’interface web est disponible en HTTPS sous argocd.kubesecurebox.com. L’accès est limité au réseau Tailscale. Vous y verrez les applications déployées, leur état de synchronisation et l’historique des déploiements.

  • Sync automatique ou manuelle
  • Self-heal : correction des dérives
  • HTTPS via cert-manager
