Portainer — Gestion du cluster Kubernetes

Portainer (Community Edition) est l'interface web pour gérer le cluster Kubernetes du KubeSecureBox. Depuis le tableau de bord, vous avez accès aux namespaces, applications, services, ingress, ConfigMaps, Secrets et volumes, ainsi qu'à un shell kubectl intégré.

Vous pouvez consulter les ressources, voir les logs des pods, lancer un exec dans un conteneur, et gérer les déploiements. Au premier accès, il faut créer le compte administrateur Portainer.

Accès

L'interface est exposée en HTTPS sous le domaine portainer.kubesecurebox.com. Premier accès : création du mot de passe admin. L’accès est réservé au réseau Tailscale.

  • Dashboard, namespaces, applications, services, ingress
  • ConfigMaps, Secrets, Volumes
  • kubectl shell et gestion du cluster
